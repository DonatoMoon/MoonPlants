# MoonPlants LSTM Forecasting

Цей модуль містить нейромережеву модель на базі **PyTorch LSTM** для прогнозування:
1. Часу до наступного поливу (в годинах)
2. Необхідної кількості води (в мілілітрах)

## Переваги над існуючим підходом (LightGBM):
- Модель автоматично аналізує часові ряди і "розуміє" швидкість висихання ґрунту (drying rate).
- Не потребує ручного створення ознак-лагів (rolling windows).
- Використовує архітектуру **Multi-task Learning** (одна модель прогнозує два параметри одночасно).

## Структура
- `data_prep.py`: підготовка даних (розрахунок цільових змінних, нормалізація, формування часових вікон).
- `model.py`: архітектура LSTM у PyTorch.
- `train.py`: скрипт для тренування моделі.
- `predict.py`: скрипт для інференсу (демонстрація використання моделі на нових або синтетичних даних).

## Як запустити
1. Встановіть залежності:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустіть тренування. Дані будуть автоматично завантажені з `../moonplants_ml/data/`:
   ```bash
   python train.py
   ```
3. Використання моделі для прогнозу. Після тренування, коли створиться папка `checkpoints/`, виконайте:
   ```bash
   python predict.py
   ```
